# Archivo de configuracion de BSPWM

![BSPWM](/imagenes/bspwm-screen.png)

## Uso

Necesitas mover el archivo bspwmrc a la carpeta .config/bspwm (En caso de que no exista, la debes crear)

```shell
mv bspwmrc ~/.config/bspwm
```
